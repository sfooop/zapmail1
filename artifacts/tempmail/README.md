# TempMail - البريد الإلكتروني المؤقت

موقع حديث وعصري لتوليد بريد إلكتروني مؤقت آمن وفعال.

## المميزات

✨ **الميزات الرئيسية:**
- توليد بريد إلكتروني مؤقت فوراً
- استقبال رسائل التفعيل الحقيقية
- واجهة مستخدم حديثة وسهلة الاستخدام
- آمن وموثوق بـ mail.gw API
- تصميم مظلم عصري مع تأثيرات مميزة

## الصفحات

1. **الرئيسية** - توليد البريد وعرض الرسائل
2. **المقالات** - مقالات مفيدة عن الخصوصية والأمان
3. **Premium** - خطط الاشتراك المدفوعة
4. **لوحة التحكم** - إدارة المحتوى والمقالات

## البيانات الافتراضية للمسؤول

**البريد:** admin@tempmail.com

## البنية

```
artifacts/tempmail/
├── src/
│   ├── pages/         # الصفحات الرئيسية
│   ├── components/    # المكونات المستخدمة
│   ├── hooks/         # React Hooks مخصصة
│   ├── store/         # حالة البيانات (Zustand)
│   ├── types/         # تعريفات TypeScript
│   ├── lib/           # المكتبات المساعدة
│   ├── App.tsx        # التطبيق الرئيسي
│   └── main.tsx       # نقطة الدخول
```

## التكنولوجيا

- **React 19** - مكتبة المستخدم
- **TypeScript** - لغة البرمجة
- **Vite** - أداة البناء
- **Tailwind CSS** - تصميم CSS
- **Framer Motion** - تأثيرات الحركة
- **Zustand** - إدارة الحالة
- **TanStack Query** - إدارة البيانات
- **Wouter** - التوجيه
- **mail.gw** - API البريد المؤقت
